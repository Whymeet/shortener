Как подключить домен {{ d.domain }}

Домен добавлен, но пока не активен — создавать на нём ссылки нельзя. Пройдите 4 шага, затем нажмите «Отметить активным» вверху страницы.

  1. Шаг 1. DNS. В панели вашего регистратора домена (где покупали {{ d.domain }}) создайте A-запись, которая направит домен на наш сервер: 
    Имя (Host):  @     (или {{ d.domain }})
Тип:         A
Значение:    {{ server_ip or "IP вашего сервера" }}
    Подождите, пока запись разойдётся (обычно несколько минут, иногда до часа), и проверьте — команда должна вернуть IP сервера: 
    dig +short {{ d.domain }}     →  {{ server_ip or "IP сервера" }}
  2. Шаг 2. nginx. На сервере добавьте домен в строку server_name (файл /etc/nginx/conf.d/nginx.short.conf), затем проверьте конфиг и перезагрузите: 
    nginx -t && systemctl reload nginx
  3. Шаг 3. HTTPS-сертификат. Выпустите TLS-сертификат (перечислите все свои домены через -d, включая новый — тогда обновится единый сертификат): 
    certbot --nginx --expand -d {{ d.domain }}
    Важно: DNS из шага 1 уже должен указывать на сервер, иначе сертификат не выпустится.
  4. Шаг 4. Активация. Вернитесь на эту страницу и нажмите кнопку «Отметить активным» вверху. После этого на {{ d.domain }} можно создавать короткие ссылки, и они будут открываться по https.
