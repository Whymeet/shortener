{# Пошаговый гайд подключения домена. Используется на дашборде (с плейсхолдером домена) и на странице домена (с реальным доменом). #} {% macro steps(domain, server_ip) %}
  1. Шаг 1. DNS. В панели регистратора домена (где покупали {{ domain }}) создайте A-запись, которая направит домен на наш сервер: 
    Имя (Host):  @     (или {{ domain }})
Тип:         A
Значение:    {{ server_ip or "IP вашего сервера" }}
    Подождите распространения (обычно минуты, иногда до часа) и проверьте — команда должна вернуть IP сервера: 
    dig +short {{ domain }}     →  {{ server_ip or "IP сервера" }}
  2. Шаг 2. nginx. На сервере добавьте домен в строку server_name (файл /etc/nginx/conf.d/nginx.short.conf), затем проверьте и перезагрузите: 
    nginx -t && systemctl reload nginx
  3. Шаг 3. HTTPS-сертификат. Выпустите TLS (перечислите все свои домены через -d, включая новый — обновится единый сертификат): 
    certbot --nginx --expand -d {{ domain }}
    DNS из шага 1 уже должен указывать на сервер, иначе сертификат не выпустится.
  4. Шаг 4. Активация. На странице домена нажмите «Отметить активным» — после этого на {{ domain }} можно создавать короткие ссылки (откроются по https).
{% endmacro %}
